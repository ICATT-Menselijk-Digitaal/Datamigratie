# Vereisten

## Registers

DMT (Data Migratie Tool) is getest in combinatie met deze registers:

* OpenZaak 1.21.1
* e-Suite 


## Data Extraction Tool (DET)

DMT communiceert niet rechtstreeks met de e-Suite. DMT is afhankelijk van de aanwezigheid van de Data Extraction Tool (DET). 
DET is een API om data uit de e-Suite op te vragen. Zie ook [de API-specificatie van DET](https://redocly.github.io/redoc/?url=https://esuite-data-extractie-gcp2.esuite-development.net/q/openapi).
